Course Overview

Labs

Lab 0 - Development Environment Setup

Ubuntu virtual machine setup and introduction to git/Github

Lab 1 - Foundations of the Web

An introduction to the HTTP protocol and web applications. Covers analytical tools such as Wireshark and telnet, developing CGI applications in C, simple PHP applications, and HTTP request structure.

Lab 2 - Front-end Web Development

Introduces the holy trinity of web development (HTML, CSS, and JavaScript) as well as Ajax, jQuery, and referencing Web APIs within your application.

Hackathons

Hackathon 1 - Cross-site Scripting Attacks and Defenses

Practice performing XSS attacks and adding defense strategies to counter XSS.

Personal Projects

Project 1 - Front-end Web Development with a Professional Profile

Developing a personal website to display information to potential employers.
